# Windows 11 Portfolio - React

A fully functional, interactive personal portfolio website inspired by the Windows 11 UI.

## Key Features
* **Interactive Taskbar & Start Menu**: Just like the real OS.
* **Draggable Windows**: Open and move project windows around.
* **Responsive Design**: Built using Tailwind CSS.
* **Contact Form**: Connected to a Node.js/MongoDB backend.

## Tech Stack
* **React.js**
* **Tailwind CSS**
* **Framer Motion** (for animations)
